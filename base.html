Les Boucles

une boucle est une structure qui permet de répéter un block de code tant qu'une condition est vraie.

C'est une façon automatique de répéter des instructions sans devoir les écrire plusieur fois

Les Types de Boucles

  • - Boucles While
  • - Boucle Do .. While
  • - Boucle For
  • - Boucle For ... oF
  • - Boucle For.. in

La boucle While

ON l'utlise quand on ne connaît pas forcément à l'avance le nombre de tours

            
                // Condition => Cette condition doit être vraie pour pouvoir continuer la boucle ( comme le if ne jamais mettre de "==" ou de "===")
                while (condition) {
                    //code à excéuter
                }

Exemple concret

            
                let n = 0;
                while (n < 6) {
                    console.log(n + "* 5 est égale à" + n * 5 );
                    n++ // veut dire n+1
                }

La Boucle For

La plus classique et la plus utilisée quand on l'avance le nombre de répétitions

            
              for (initialisation; condition; incrémentation) {
                // code à exécuter
                for (let i = 0; i < 5; i++) {
                console.log("compteur :" + i);
                }

Exemple concret

Une boucle for permet de traverser un tableau à partir de ses index

            
                let voitures = ["Dacia", "Lada", "Audi", "Daihatsu", "Toyota", "Chevrolet"]

                for(let i =0; i< voitures.length; i++){
                console.log(voitures[i])
                }
