1. Introduction à Javascript

C'est quoi le Javascript

Le javascript est un language de programmation qui donne de l'intéractivité aux pages web.

  • - HTML structure ou le squelette
  • - CSS = style (les couleurs, la mise en pages etc)
  • - JS = logique (les réactions à un utilisateur, des calculs, et la dynamique de la page)

Où s'exécute le JS ?

  • - Dans le navigateur (Chrome, Firefox, Safari etc)
  • - Sur un seveur qui tourne avec Node.js -> il ne sera plus un langage Front-End mais Back-End comme le PHP ou le Python etc...

A retenir : même langage, mais deux environnements différents

Premier pas : console, balise script

  1. Ouvrir votre navigateur préféré
  2. Click droit -> inspecter, ou si vous êtes sur windows F12
  3. Allez dans l'onglet console
