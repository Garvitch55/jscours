Les fonctions

Une fonction est un bloc de code réutilisable qui effectue une tâche précise

Elle permet d'organiser ton code, et d'éviter les répétitions, et rend ton code plus lisible

                    //instructions à exécuter
                }

                // fonction principale qui va calculer le prix total d'un panier
                function calculatePriceCart(cart)  {
                    //avant de calculer on va initialisé le total qui pourrait être 0.
                    let total=0;

                    for (let i=0; i < cart.length; i++){
                        // pour faciliter la lecture on renomme élément acutuel de la boucle par
                        // variable plus simple à écrire
                        const item= cart[i];
                        // le retour de la fonction multiply "injecte" son résultat dans itemPriceTotal
                        const itemPriceTotal = multiply (item.price, item.quantity);
                        total = add(total, itemPriceTotal)
                    }
                    return total
                    
                }

Les arguments

Un argument est une valeur qu'on donne à une fonction quand on l'appelle, pour qu'elle travaille avec cette valeur

  • Quand on déclare la function, on parle de paramètres
  • Quand on appel la fonction on parle d'arguments
                
                    function bonjour (prenom) {
                        console.log("Bonjour" +prénom);
                    }

                    bonjour ("Jean-michel");

Plusieur Arguments

Une fonction peut avoir autant d'argument que vous voulez et tu dois passer le même nombre d arguments quand tu l'appelles

Si le nombre d'argument est trop ou qu'ils en manquent, la fonction retourne undefined.

                
                    function add(nbrA, nbrB) {
                    return nbrA + nbrB;
                }
                console.log(add(7, 19)); //26

Valeurs par défaut

                
                    function bonjour (prenom= "Sébastien") {
                        console.log("Bonjour" +prénom);
                    }

                    bonjour ();// "bonjour Sébastien"
